# AI Development Assistant

An Electron-based application that creates isolated development environments using Docker containers and integrates with local AI models for automated development tasks.

## Features

- Create isolated Docker containers for each project
- Chat with local AI (Ollama) to generate code
- Real-time preview of your application
- Console output for Docker and application logs
- GitHub publishing capabilities

## Prerequisites

- Node.js 16+
- Docker Desktop
- Ollama (with qwen3-coder model)
